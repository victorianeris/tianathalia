*Descrição Técnica do Projeto e Arquitetura de Tecnologias*

Este projeto de Front-End foi desenvolvido com o objetivo de criar uma interface moderna, responsiva e semântica. O processo de desenvolvimento uniu práticas fundamentais de codificação web com a engenharia de prontos (prompt engineering) utilizando o Gemini IA como assistente de codificação.

Abaixo estão detalhadas as tecnologias utilizadas, suas respectivas funções e como a Inteligência Artificial foi integrada ao fluxo de trabalho.

*Tecnologias de Desenvolvimento (Core Front-End)*
- HTML5 (HyperText Markup Language): * Função: Utilizado para construir a estrutura esquelética e a semântica da página.

Práticas aplicadas: Foco no uso de tags semânticas (como <header>, <main>, <section>, <footer>) para garantir a acessibilidade (SEO) e uma estrutura limpa para leitores de tela.

- CSS3 (Cascading Style Sheets):

Função: Responsável pela estilização visual, identidade visual, cores e layout da aplicação.

Práticas aplicadas: Uso de metodologias modernas de layout como Flexbox e/ou CSS Grid para garantir o alinhamento dos elementos. Implementação de Media Queries para criar um design responsivo, garantindo que a página se adapte perfeitamente a dispositivos móveis, tablets e desktops.

- Integração e Participação da Inteligência Artificial (Gemini IA)
O Gemini IA atuou de forma colaborativa durante todo o ciclo de desenvolvimento do código através de técnicas de Pair Programming (Programação em Dupla). A IA foi utilizada para:

Arquitetura e Estruturação de Layout: Auxílio na tomada de decisão sobre a melhor estrutura de tags HTML para layouts complexos.

Resolução de Problemas (Debugging): Identificação rápida de falhas de estilização no CSS (como quebras de layout em telas menores) e correção de bugs de especificidade.

Refatoração de Código: Otimização das regras de CSS para tornar o arquivo mais limpo, legível e com menor consumo de linhas de código.

Sugestão de Boas Práticas: Implementação de padrões modernos de acessibilidade e design system (combinação de paleta de cores e tipografia).

*Ferramentas de Suporte e Versionamento*
Git: Ferramenta de controle de versão utilizada localmente para rastrear o histórico de alterações do código e gerenciar o fluxo de trabalho.

GitHub: Plataforma de hospedagem de código utilizada para armazenar o repositório remoto, permitindo a exibição pública do portfólio e garantindo o backup seguro do projeto.
